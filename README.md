#### Criando projeto

* 1. comando no terminal
```  
npx create-next-app@latest app1
```

* 2. entrando na pasta do projeto
 ```
 cd app1
 ```

 * 3. rodar o projeto
 ```
 npm run dev
 ```

 ### Github Salvar

> #### Criando pela primeira vez
 * 1. Iniciar um repo
 ```
 git init
 ```

 * 2. Adicionar arquivos
 ```
 git add .
 ```
